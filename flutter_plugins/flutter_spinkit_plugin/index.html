Spinkit Plugin

A collection of loading indicators animated with flutter.

Installation

Let’s look at different ways to install and use the flutter spinkit plugin

Add dependency

Add this to your package’s pubspec.ymal file:

    dependencies:
        flutter_spinkit: ^5.2.0

Install package from command line

    flutter pub add flutter_spinkit

Import Package

    import 'package:flutter_spinkit/flutter_spinkit.dart';

Usage

Add these to any widget as a child:

    SpinKitRotatingCircle(
        color: Colors.white,
        size: 50.0,
    ),

    SpinKitDoubleBounce(
        color: Colors.white,
        size: 50.0,
    ),

    SpinKitWave(
        color: Colors.white,
        size: 50.0,
    ),

    SpinKitWanderingCubes(
        color: Colors.white,
        size: 50.0,
    ),

There are many available widgets can be seen attached here: Spinkit on pub.dev .
