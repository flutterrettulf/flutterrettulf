What is just_audio Plugin for Flutter

just_audio is a feature-rich audio player for Android, iOS, macOS, web, Linux and Windows.

Installation

Add dependency

Add this to your package’s pubspec.ymal file:

    dependencies:
        just_audio: ^0.9.29

Install package from command line

    flutter pub add just_audio

Import Package

    import 'package:just_audio/just_audio.dart';

Platform specific configuration

Android

Add the following to your AndroidManifest.xml file to enable your app to access audio files online.

    <uses-permission android:name="android.permission.INTERNET" />

Add the following attribute to the application element if you want to connect to non-HTTPS URLs or if you employ a functionality that relies on the proxy, including headers, caching, or streaming audio sources:

    <application
        android:usesCleartextTraffic="true"
        ...>
    </application>

iOS

If you wish to connect to non-HTTPS URLS, or if you use a feature that depends on the proxy such as headers, caching or stream audio sources, add the following to your Info.plist file:

    <key>NSAppTransportSecurity</key>
    <dict>
        <key>NSAllowsArbitraryLoads</key>
        <true/>
    </dict>

macOS

Add the following to your DebugProfile.entitlements and Release.entitlements files to allow your macOS programme to access audio files on the Internet:

   <key>com.apple.security.network.client</key>
    <true/>

Linux

You may enable Linux support by including a second dependency in your pubspec.yaml file in addition to just audio.

dependencies:
  just_audio: any # substitute version number
  just_audio_mpv: any # substitute version number

Windows

You may enable Windows support by including a second dependency in your pubspec.yaml file in addition to just audio.

dependencies:
  just_audio: any # substitute version number
  just_audio_windows: any # substitute version number

Usage

Add these to any widget as a child:

import 'package:just_audio/just_audio.dart';
final player = AudioPlayer();                   // Create a player
final duration = await player.setUrl(           // Load a URL
    'https://foo.com/bar.mp3');                 // Schemes: (https: | file: | asset: )
player.play();                                  // Play without waiting for completion
await player.play();                            // Play while waiting for completion
await player.pause();                           // Pause but remain ready to play
await player.seek(Duration(second: 10));        // Jump to the 10 second position
await player.setSpeed(2.0);                     // Twice as fast
await player.setVolume(0.5);                    // Half as loud
await player.stop();                            // Stop and free resources

Supported features

Audio sources

The following audio sources are supported:

  • Remote URLs (HTTP, HTTPS, FTP, SFTP, and others)
await player.setUrl('https://foo.com/bar.mp3');
  • Local files
await player.setFilePath('/path/to/file.mp3');
  • Assets
await player.setAsset('assets/audio.mp3');
  • Byte buffers 
    // Feed your own stream of bytes into the player
class MyCustomSource extends StreamAudioSource {
  final List<int> bytes;
  MyCustomSource(this.bytes);

  @override
  Future<StreamAudioResponse> request([int? start, int? end]) async {
    start ??= 0;
    end ??= bytes.length;
    return StreamAudioResponse(
      sourceLength: bytes.length,
      contentLength: end - start,
      offset: start,
      stream: Stream.value(bytes.sublist(start, end)),
      contentType: 'audio/mpeg',
    );
  }
}

await player.setAudioSource(MyCustomSource());
player.play()

Playback controls

The following playback controls are supported:

  • Play
player.play();
  • Pause
player.pause();
  • Stop
player.stop();
  • Seek
player.seek(Duration(seconds: 10));
  • Seek to next/previous
player.seekToNext();
player.seekToPrevious();
