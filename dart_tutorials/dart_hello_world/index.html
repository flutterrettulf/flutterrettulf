Dart Hello World Program

Updated on October 21, 2022
  • TL;DR
  • Download & Install the Dart SDK
  • Dart Hello World
  • Summary
  • References
  • Dart Tutorials

TL;DR

  • Install Dart SDK from here.
  • Create a file named hello_world.dart and add the following code: 
    void main() { 
  print('Hello World');
}
  • Run the following command to compile the code: 
    dart hello_world.dart
  • You should see the following output: 
    Hello World

Download & Install the Dart SDK

The Dart Software Development Kit (SDK) is a set of libraries and tools that allow you to develop Dart applications. You can use a package manager to easily install and update a stable channel Dart SDK. The Dart SDK is available for Windows, macOS, and Linux. You can download the Dart SDK from the official website.

To verify the Dart SDK installation, you can run the following command in a terminal:

dart --version

Dart Hello World

To create a Dart program, you need to create a file with the extension .dart. Let’s create a file named hello_world.dart. You can save the file in any directory. For example, you can create a directory named dart_tutorials and save the file in that directory.

void main() {
    print('Hello World');
}

In the above code, we use the print() function to print the string Hello World to the console. The main() function is the entry point of the Dart program. The void keyword specifies that the function does not return a value. Every dart app must have a main() function. You can write any number of functions in a Dart program. The main() function is the first function that is executed when the program starts. Other functions are executed in the order they are called in the main() function.

To run the Dart program, you can use the dart command. You can run the program using the following command:

dart hello_world.dart

The output of the program will be:

Hello World

Summary

In this tutorial, we learned how to install the Dart SDK and write a simple Dart program to print Hello World on the console. We also learned about the main() function in Dart. In the next tutorial, we will learn about the Dart data types.
