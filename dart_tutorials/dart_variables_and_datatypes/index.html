What are variables in Dart?

If you are familiar with programming, you would be aware of variables. A variable is basically a reserved memory location to store values. In other words, a variable in a program is a name given to a memory location that can store some data.

For example, let’s say we want to store the temperature of a particular city in Celsius. In this case, we would create a variable, temperature, and store the value of 20 in it. Similary we can store other informations such as the name of the city (hyderabad) and the state (telangana) in variables. Let’s see a few examples below.

    int temperature = 20; // temperature of hyderabad is stored
    String name = "hyderabad"; // store the city name
    String state = "telangana"; // store the city state

Syntax for Declaring a Variable

In order to declare a variable in Dart, we simply write <data_type> <variable_name>. For example, the code for the declaration of the variable temperature would be as follows:

    int temperature;

Let’s look at the syntax in detail:

Part of the statement Details
Variable type The type of variable will be int, i.e, an integer value.
Variable name The name of the variable is temperature. The naming convention of variables in Dart is similar to the naming convention followed by other programming languages. We will go over these naming conventions in the next section.

Naming Conventions for Dart Variables

There are a few rules followed when naming variables in Dart. Following are some of the naming conventions we can follow;

  • It is recommended to follow CamelCase, i.e, thisIsACamelCaseVariable.
  • You should not start a variable name with special characters.
  • You can not use reserved keywords in Dart while naming a variable. For example, using the keyword void for the variable that stores a double value is not allowed.
  • The variable name should be descriptive, i.e, it should represent the task for which it is used. it is true that we can create a variable named a to store the number 2, but it i won’t help anyone in understanding the code. So it is better to write variable names that are descriptive.
  • Variable name or identifiers can’t begin with number.

Data Types in Dart

The type of the data which a variable is used to store is known as the data type of that variable. Data types are used when mapping data types to a specific memory location. Different variables can contain values of different data types. In other words, the data type of a variable determines the size and the range of values that that variable can hold.

Type Safety

Dart is type-safe programming language, meaning that not all types are compatible. If you declare a variable as an int, then the compiler ensures that the variable is type-safe meaning on runtime, other types of data cannot be inserted into the declaration. To see this in action, let’s take a look at the code below:

    int age = 18;
    age = 19.2; // Will throw an error

If you pass anything other than int (for example double), then the compiler would output an error.

    ...(int).dart:8:7: Error: A value of type 'double' can't be assigned to a variable of type 'int'.
    Try changing the type of the left hand side, or casting the right hand side to 'int'.
         age = 19.2;

Types of Data Types in Dart

Dart supports the following data types:

  • Numbers (int and double)
  • Strings (String)
  • Booleans (bool)
  • Lists (list)
  • Maps (map)
  • Dynamic (dynamic)

Let us check out each of these datatypes one at a time.

Numbers

The number data types represent numbers ie. positive (+) and negative (-) in the arithmetic expression. Numbers are of two types int and double. Integers represent non-fractional numbers and floating-point numbers types are called double. The Double data-type represents 64 bit (double-precision) floating-point numbers.

    int x = 1;
    double y = 1.1;

Strings

The string type represents a sequence of unicode characters. Strings are usually used to store data such as name, address, etc… A Dart string is a sequence of UTF-16 code units.

The String keyword is used to declare a string variable.

    String name = "ali";

Booleans

The boolean data type represents the logical values that are just two - true or false. Booleans are mostly used to store conditions, such as if any condition is true or false.

    bool gameOver = true;

Lists

Lists are used to hold a collection of objects. Lists are similar to arrays that are typically used in other programming languages. Dart has support for generic classes. This means that we can specify the type of data that a list can store. Take a look at the code below

    List<String> names = ['ali', 'bharat', 'sam', 'tom'];
     List<int> ages = [18, 25, 32];
    List<String> cities = ["hyderabad", "chennai"];

The first list, names, consists of strings and can have other strings inserted into it. The second list, ages, consists of integers, and can have integers inserted into it. And, the third list, cities, consists of strings and can have other strings inserted into it. Note that we have used square brackets ([]), as it is used to define a list.

Maps

Maps are also called associative arrays. A map is an object that associates keys and values. A key can be any valid string, integer or symbol while the value can be any valid dart object. We can add new entry to a map programmtically.

    Map<int, String> countryIdToName = {1: "India", 2: "America"};

The dart:core library enables creation and manipulation of these collections through the predefined List and Map classes respectively.

Dynamic

Dart has a special data-type called dynamic which is useful when the type of the variable is unknown during the compile-time. We can use the dynamic data-type as illustrated below:

   dynamic n = 22.3;
   n= "hi"; // value changed to String
   n = true; // value changed to Bool
   print(n); // print Bool

Summary

We have learnt how to declare the veriables and the types of data that can be stored in them. Dart is a type safe programming language meaning that one cannot insert another kind of data-type into a variable declaration. Instead, compiler throws the error. We learnt about different types of data such as numbers, strings, integers, double, dynamic, maps etc.
