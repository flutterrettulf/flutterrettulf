Introduction

Hello and welcome to our tutorial on using lists in Dart. In this article, we’ll go through several ways to work with lists in Dart. Every programming language needs lists as a fundamental data structure, and Dart offers a variety of effective ways to interact with them. We will examine these techniques in-depth in this article. We hope this article will be helpful to you whether you’re a new or seasoned coder. Let’s get going!

How to insert elements to a list in dart?

Dart provides several methods to insert elements into a list. They are:

  1. add()
  2. addAll()
  3. insert()
  4. insertAll()

1. add()

The add() method appends the specified value to the end of the list. The list must be growable, when using the add() method.

final numbers = [1, 2, 3, 6];
numbers.add(7);
print(numbers); // [1, 2, 3, 6, 7]

In the above example, the numbers list has the values [1,2,3,6]. Then we call the add() method with the value 7 as the parameter. The numbers list is updated by appending the value 7 to the list.

2. addAll()

The addAll() is used when we want to append multiple items to a list. The addAll() method accepts an iterator as the parameter and appends them to the end of the list. This method extends the length of the list by the number of objects in iterable, So the list must be growable.

final ls = [1, 2, 3, 4, 5];
ls.addAll([6, 7, 8]);
print(ls); // [1, 2, 3, 4, 5, 6, 7, 8]

3. insert()

insert() method accepts two parameters: index and the object to be inserted. It increases the length of the list by one and shifts all objects at or after the index towards the end of the list.The index value must be greater than or equal to 0 and no greater than length.

final numbers = <int>[1, 2, 3, 4];
const index = 2;
numbers.insert(index, 10);
print(numbers); // [1, 2, 10, 3, 4]

4. insertAll()

insert() method accepts two parameters: index and an iterable of objects to be inserted.It increases the length of the list by the length of iterable and shifts all later objects towards the end of the list, So, the list must be growable. The index value must be greater than or equal to 0 and no greater than length.

final numbers = <int>[1, 2, 3, 4];
final insertItems = [10, 11];
numbers.insertAll(2, insertItems);
print(numbers); // [1, 2, 10, 11, 3, 4]

How to remove elements from a list in dart?

There are several ways to remove elements from a list. You can remove one element from the list by calling remove() or clear the whole list by calling clear(). Let’s take a detailed look at different ways to remove elements from a list in dart.

  1. clear()
  2. remove()
  3. removeAt()
  4. removeLast()
  5. removeRange()

1. clear()

clear() method is used to remove all elements from a list. In other words, empty the list.

final numbers = <int>[1, 2, 3];
numbers.clear();
print(numbers.length); // 0
print(numbers); // []

2. remove()

Removes the first occurrence of value from this list. If the value was in the result, this function returns true and removes the first occurrence of value from the list. If the value was not in the list, this function returns false. The list must be growable.

final parts = <String>['head', 'shoulders', 'knees', 'toes'];
final retVal = parts.remove('head'); // true
print(parts); // [shoulders, knees, toes]

final retVal = parts.remove('head'); // false
print(parts); // [shoulders, knees, toes]

3. removeAt()

The removeAt() method is used to remove the object at the specified index. This method removes the object at the specified index, moves all later elements down by one and reduces the length of the list by one. Returns the removed value. The index must be in the range 0 ≤ index < length.

final parts = <String>['head', 'shoulder', 'knees', 'toes'];
final retVal = parts.removeAt(2); // knees
print(parts); // [head, shoulder, toes]
print(retVal); // knees
print(parts.length); // 3

4. removeLast()

Removes and returns the last object in this list. The list must be growable and non-empty.

final parts = <String>['head', 'shoulder', 'knees', 'toes'];
final retVal = parts.removeLast(); // toes
print(parts); // [head, shoulder, knees]
print(parts.length); // 3

5. removeRange()

The removeRange() method is used to remove a range of objects from the list.The method accepts two arguments: start and end. The first argument is the index of the start of the range. The second argument is the index of the end of the range. Every object with positions greater than or equal to start and less than end, will be removed from the list. The object at the index end will not be removed.

A range from start to end is valid if 0 ≤ start ≤ end ≤ length. An empty range (with end == start) is valid.

final numbers = <int>[1, 2, 3, 4, 5];
numbers.removeRange(1, 4);
print(numbers); // [1, 5]

How to remove elements from a list in dart, based on a condition?

Earlier, we looked at removing elements a specific index or a range. Next, we are going to look at how to remove elements from a list based on some conditions. Dart provides some usefull methods for this purpose.

  1. removeWhere()
  2. retainWhere()

1. removeWhere()

Removes all objects from this list that satisfy the provided condition. The removeWhere() method accepts a function as an argument(test function). Each item in the list will be passed to the test function as an argument. The test should perform the required checks and return true or false. All the objects where the test function returns true will be removed from the list.

final numbers = <String>['one', 'two', 'three', 'four'];
numbers.removeWhere((item) => item.length == 3);
print(numbers); // [three, four]

2. retainWhere()

Removes all objects from this list that does not satisfy the provided condition. The working of the retainWhere() method is similar to that of removeWhere() except that it retains all the objects where the test function returns true.

final numbers = <String>['one', 'two', 'three', 'four'];
numbers.retainWhere((item) => item.length == 3);
print(numbers); // [one, two]

How to find if an item is present in the list?

We have several options in dart, to check if the list contains a specific item.

  1. indexOf()
  2. indexWhere()
  3. lastIndexOf()

1. indexOf()

Searches the list from index start to the end of the list. The first time an object o is encountered so that o == element, the index of o is returned. If the element is not present, this method returns -1.

final notes = <String>['do', 're', 'mi', 're'];
print(notes.indexOf('re')); // 1
final indexWithStart = notes.indexOf('re', 2); // 3
final index = notes.indexOf('fa'); // -1

2. indexWhere()

This method accepts a test function as a parameter. Searches the list from index start to the end of the list. The test function will be called each time with the element as the parameter. The test function should return true or false after performing required checks. The first time an object o is encountered so that test(o) is true, the index of o is returned. If the element is not present, this method returns -1.

final notes = <String>['do', 're', 'mi', 're'];
final first = notes.indexWhere((note) => note.startsWith('r')); // 1
final second = notes.indexWhere((note) => note.startsWith('r'), 2); // 3
final index = notes.indexWhere((note) => note.startsWith('k')); // -1

3. lastIndexOf()

Searches the list backwards to index 0. The method accetps an optional parameter start. If the start index is given, the method searhes the list backwards from start index to 0. if `start is not given, it begins the search from the last element.

The first time an object o is encountered so that o == element, the index of o is returned.

final notes = <String>['do', 're', 'mi', 're'];
const startIndex = 2;
final index = notes.lastIndexOf('re', startIndex); // 1, with startIndex as 2
final index = notes.lastIndexOf('re'); // 3, without startIndex. default=index of last element
final index = notes.lastIndexOf('fa'); // -1

How to sort a list in dart?

Dart provides a sort() method to sort the values in a list. The sort() method accepts a compare function as its parameter. We can provide a custom logic on how to sort the list. For example, we could provide a compare function that checks the alphabetical order, or may be the length of the elements etc… The compare function must act as a Comparator.

final numbers = <String>['two', 'three', 'four'];
// Sort from shortest to longest.
numbers.sort((a, b) => a.length.compareTo(b.length));
print(numbers); // [two, four, three]

The default List implementations use Comparable.compare if compare is omitted.

final numbers = <int>[13, 2, -11, 0];
numbers.sort();
print(numbers); // [-11, 0, 2, 13]

In that case, the elements of the list must be Comparable to each other.

A Comparator may compare objects as equal (return zero), even if they are distinct objects. The sort function is not guaranteed to be stable, so distinct objects that compare as equal may occur in any order in the result:

final numbers = <String>['one', 'two', 'three', 'four'];
numbers.sort((a, b) => a.length.compareTo(b.length));
print(numbers); // [one, two, four, three] OR [two, one, four, three]

How to reduce a collection in dart to a single value: reduce() method

Reduces a collection to a single value by iteratively combining elements of the collection using the provided function.The iterable must have at least one element. If it has only one element, that element is returned.

final numbers = <double>[10, 2, 5, 0.5];
final result = numbers.reduce((value, element) => value + element);
print(result); // 17.5

The reduce method takes a function as an argument, which is applied to each element in the list. In this case, the function takes two arguments value and element and returns the sum of those two numbers. The reduce method applies this function to the elements of the list, starting with the first two elements and then combining the result with the next element and so on, until the entire list has been reduced to a single value. In this example, the list is reduced to the sum of all its elements.

Conclusion

We covered the add method, which allows you to append an element to the end of a list, and the insert method, which allows you to add an element at a specific index in the list.
