What are Operands & Operators in Dart?

In mathematics, an operator is a mathematical symbol or function that indicates the operation to be done on a value or a set of values. This is similar to how an operator in programming works. An operator defines how the operands will be processed.

Consider a simple expression val3 = val1 + val2. Here, val1 and val2 are the operands, and + is the operator. The operator receives the operands and performs an operation on them. The operator returns the result of the operation to the calling code. Here the operator returns the sum of val1 and val2. The val3 is the variable in which the result of the operator is stored.

Bitwise Operators in Dart

In Dart, you can use bitwise operators to perform operations on bits. The following table shows the bitwise operators:

Operator Meaning Syntax Explanation
| OR A | B Set each bit to 1 if both bits are 1.
& AND A & B Set each bit to 1 if both bits are 1.
^ XOR A ^ B Set each bit to 1 if only one of two bits is 1.
« left shift A«z Shift a bit pattern to the left, and fill the low-order bits with 0.
>> right shift A»z Shift a bit pattern to the right, and fill the high-order bits with 0.
~ one’s complement ~A Invert all bits. This operation normally means “not A”.

Logical Operators in Dart

In Dart, you can use following operators to perform logical operations, such as and, OR or NOT. The following table shows the logical operators:

Operator Meaning Syntax Explanation
&& AND A && B Only if A and B are true, results true, otherwise results false.
|| OR A || B If A or B, or A and B, results true.
! NOT !A If A is false, results true, otherwise results false.

Assignment Operators in Dart

Dart also supports following assignment operators, which are used to assign a value to a variable or expression.

Operator Explanation Syntax Example Output
= The assignment assigns the value of its right-hand operand to its left-hand operand. The right-hand operand is first evaluated, and then the assignment is performed. Assigning a value of a lower-precedence operator to a variable or field of higher precedence results in an error. A = B i = 10 10
+= Add and assignment (also known as Compound assignment operator) operator adds the value of its right-hand operand to the left-hand operand and assigns the result value to the left-hand operand. A += B; A = A + B; i += 10 110
-= Subtracts and assigns right-hand operand from the left-hand operand and assigns the result to the left-hand operand. A -= B; A = A – B; i += 10 -90
